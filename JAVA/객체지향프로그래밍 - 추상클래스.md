## 추상클래스

https://limkydev.tistory.com/188		-- 참고 링크

**1. 추상클래스는 하나의 추상메소드를 가지고 있어야 한다.**

**2. 추상클래스는 인스턴스 생성이 불가능함(new 키워드 사용 불가능)**

- **인스턴스 생성을 하려면, 자식클래스를 이용해야함**

**3. 추상 메소드를 포함하는 클래스는 반드시 추상클래스로 정의되야함**
